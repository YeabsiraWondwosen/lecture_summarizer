# Automatic Lecture Summarization System

A **Python-based GUI application** that automatically summarizes long lecture notes. The system uses **NLP techniques** (tokenization, stop-word removal, word frequency analysis) to extract the most important sentences from a lecture. Built with **Tkinter** for a user-friendly interface.

---

## Features

- Accepts **long lecture text** input directly in the GUI.
- Automatically generates a **concise summary**.
- Adjustable **summary length (%)**.
- Scrollable input and output text areas for easy viewing.
- Works on **Python 3.13+** without extra dependencies (no `punkt_tab` issue).
- Safe for university projects and easy to demo in viva.
